# Coffee Shop Sales Analysis ☕

## 📊 Project Overview
This project analyzes coffee shop sales data using Excel to generate insights on revenue, product performance, and customer behavior.

## 🔑 Key KPIs
- Total Revenue
- Average Order Value (AOV)
- Top Selling Products
- Sales by Time (Hourly/Monthly)
- Category-wise Sales

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

## 📈 Insights
- Identified peak sales hours
- Found top-performing products
- Analyzed sales trends over time
